前言

2004年，google一篇MapReduce: Simplified Data Processing on Large Clusters论文开启了大数据时代，今天我们要讲的javascript自带的map/reduce高阶函数，与这篇论文中的mapreduce概念相同。

map

如果我们要对一个Array数组里的每个元素采取相同的操作，我们可以使用map函数返回结果Array

var arr=[1,2,3,4,5];
arr.map(pow);

map()中传入的参数是pow，即函数对象本身
我们同样可以把数字数组转化成字符串
arr.map(String);

reduce

reduce对Array进行迭代计算，作为reduce参数的函数必须有两个参数，每次将前面的计算结果与下一个元素运算。
比如：

var x=[x1,x2,x3,x4];
x.reduce(f);//=f(f(f(x1,x2),x3),x4);

总结

利用javascript提供的map/reduce函数可以组合成许多的十分简洁的函数实现，下次再找几个例子慢慢分析。
